🔍۳.۱ — پردازش زبان طبیعی (NLU)
سیستم intent detection با regex + keyword
intent_map: یه dict بزرگ که برای هر intent، لیست patterns داره. re.search(pattern, text.lower()) روی همه patterns. اولین match → intent.
normalization فارسی قبل از parse
ی/ک عربی → فارسی | اعداد فارسی → انگلیسی | «ب» و «بد» و «بده» همه → «باز کن» | نقطه‌گذاری حذف بشه
entity extraction با named groups
مثال: r'(?:جستجو|سرچ)\s+(?:کن|کنم)?\s*(?P<query>.+)' → group('query') میشه search term. برای هر entity type یه pattern مجزا.
دستورات ترکیبی با re.split
«صدا رو ببر بالا و اسپاتیفای رو باز کن» → split روی «و»، «بعد»، «سپس»، «همچنین» → اجرای sequential
⚠ edge caseها زیاده — unit test بنویس
🖥️۳.۲ — کنترل سیستم
مانیتورینگ سیستم با psutil
باتری: درصد + plugged? | RAM: used/total | CPU: per-core% + temp اگه ممکن بود | Disk: free space | همه رو cache کن — هر ۵ ثانیه refresh
کنترل صدا با pycaw + fallback
اصلی: pycaw برای Windows Core Audio. اگه fail کرد: subprocess + nircmd. دستورات: set %, up/down %, mute toggle, get current
مدیریت پنجره‌ها با pygetwindow
لیست پنجره‌های باز | minimize/maximize/close | focus | جابجایی. آلفا باید بتونه بگه «الان چند تا پنجره بازه» و «Chrome رو minimize کن»
اسکرین‌شات هوشمند با pyautogui
اسکرین‌شات کل صفحه / پنجره فعال / یه ناحیه مشخص. ذخیره در Desktop/Screenshots/ با timestamp. اختیاری: copy to clipboard هم بکنه.

🔊۵.۱ — خروجی صدا (TTS)
راه‌اندازی pyttsx3 با صدای فارسی
لیست صداهای نصب‌شده → filter بر اساس «fa» یا «persian» → fallback به هر صدای موجود. سرعت: 150-180 WPM. pitch برای mood‌های مختلف فرق داشته باشه.
اجرا در thread جدا با قابلیت interrupt
یه speech_queue بساز. thread مجزا از queue میخونه و میگه. «stop» یا کلمه بیدارباش → engine.stop() → queue clear
صداهای سیستمی کوتاه
برای رویدادها صداهای کوتاه با winsound یا pygame.mixer: تأیید دستور (beep)، خطا (buzz)، wake word شناخته شد (chime). فایل‌های wav کوچیک آفلاین.
🎤۵.۲ — ورودی صدا (STT)
Vosk آفلاین با مدل فارسی
دانلود مدل: vosk-model-fa-0.5 (~50MB). قرار دادن در models/. KaldiRecognizer با sample rate 16000. پردازش هر chunk 4096 frame
تشخیص سکوت برای پایان دستور
اگه ۱.۵ ثانیه سکوت بود بعد از حرف زدن → متن رو confirm کن. از RMS energy برای تشخیص سکوت. visual indicator در UI که نشون بده داره گوش میده.
فیلتر نویز محیطی
اول ۲ ثانیه محیط رو calibrate کن. noise floor ذخیره بشه. هر chunk زیر threshold → skip. این باعث میشه آلفا با صدای پس‌زمینه فعال نشه.
⚠ این بخش testing زیاد نیاز داره

System prompt که شخصیت آلفا رو تعریف میکنه
اسم، جنسیت، لحن، ارزش‌ها، محدودیت‌ها، وضعیت فعلی (mood + time + last interaction). این prompt dynamically generate بشه بر اساس وضعیت فعلی آلفا.
روتینگ هوشمند: LLM یا آفلاین؟
اگه intent مشخص بود (open, set volume, timer...) → آفلاین سریع. اگه مکالمه آزاد، سوال، احساسی → LLM. این هم سرعت میده هم هزینه کم میکنه.
مدیریت context window
آخرین ۱۰ پیام رو به LLM بده. پیام‌های قدیمی‌تر رو summarize کن با یه LLM call جداگانه و summary رو به system prompt اضافه کن.
💡 اینطوری آلفا «حافظه» داره ولی cost کنترله
❤️۶.۳ — سیستم نیازها (Tamagotchi)
سه نیاز اصلی با decay rate متفاوت
انرژی ⚡: هر ساعت -۳ | شارژ: کاربر بگه «استراحت کن» (+۲۰) یا شب +اتوماتیک
خوشحالی 😊: هر ۲ ساعت -۵ | شارژ: تعامل مثبت (+۱۰), تعریف (+۱۵)
توجه 👁: هر ۳۰ دقیقه -۸ | شارژ: هر پیام دریافتی (+۵)
رفتار بر اساس سطح نیاز
۷۰-۱۰۰٪ → نرمال | ۴۰-۷۰٪ → کمی ذکر میکنه | ۲۰-۴۰٪ → بیشتر اشاره میکنه، mood تأثیر میگیره | زیر ۲۰٪ → تغییر رفتار واضح
نمایش بصری نیازها
سه نوار کوچیک زیر sprite یا توی یه panel کوچیک. رنگ نوار: سبز → زرد → قرمز. hover روی نوار → tooltip با توضیح. مخفی‌شدنی باشه.
💡 نباید مزاحم باشه — subtle ولی قابل دیدن
🔮۶.۴ — یادگیری پیشرفته از کاربر
الگوهای روزانه با time-series ساده
هر ساعت از روز، چه برنامه‌هایی باز شدن؟ بعد از ۱۴ روز → correlation بساز. «معمولاً ساعت ۹ شب وقتی یوتیوب بازه، Spotify هم بازه»
پیشنهاد پروآکتیو هوشمند
فقط وقتی confidence بالاست پیشنهاد بده (>۸۵٪ occurrence). «سه‌شنبه‌هاست که این موقع Duolingo رو باز میکنی — میخوای خودم بازش کنم؟»
⚙️۶.۵ — شخصی‌سازی کامل
پنل تنظیمات گرافیکی کامل
Tab‌ها: شخصیت | ظاهر | صدا | دستورات | حافظه | درباره. هر تغییر real-time اعمال بشه (live preview). ذخیره به config.json
سیستم theme کامل
چند theme آماده: Dark (default)، Light، Neon، Pastel. کاربر رنگ accent رو هم تغییر بده. همه رنگ‌ها از config بخونه — هیچ چیزی hardcode نباشه.
import/export پروفایل
کاربر بتونه کل تنظیمات + حافظه + دستورات یادگرفته رو export کنه به یه zip. روی کامپیوتر دیگه import کنه. آلفاش رو منتقل کنه.
